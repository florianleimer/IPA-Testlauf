# IPA-Testlauf
IPA-Testlauf - M306 &amp; M326

## Project setup
```
npm install
```
```
php api/composer.phar install -d api/
```

### Run project
```
Run XAMPP (Apache & MySQL)
```
```
Change apiUrl in vue.config.js to your needs
```
```
Run 'npm run serve'
```
